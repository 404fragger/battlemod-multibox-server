## Setup
1. Run `npm install` to install dependencies
2. Run `npm run start` to start the server

## New Setup - Windows
1. Just run the `battlemod-multibox-server-win.exe` (no installation required)

## New Setup - MACOS
1. Just run the `battlemod-multibox-server-macos` (no installation required)

## New Setup - Linux
1. Just run the `battlemod-multibox-server-linux` (no installation required)
